# 内容简介

本书以"本地大模型驱动的 RAG 全栈应用"为主线，围绕电商工单智能检索这一贯穿案例，系统讲解从 Ollama 本地部署、Embedding 向量化、ChromaDB 向量检索、FastMCP Agent 工具封装，到 FastAPI SSE 流式后端与 Next.js 聊天前端的完整链路实现。

全书共 10 章。第 1、2 章介绍 Ollama 本地大模型的快速上手与 HTTP API 流式响应；第 3、4 章讲解文本切片与 Embedding 向量化的工程要点；第 5、6 章带读者搭建 ChromaDB 向量数据库与工单知识库；第 7 章用 FastMCP 把检索能力封装为 Agent 标准工具；第 8、9 章实现 FastAPI 流式 RAG 后端与 Next.js 前端聊天界面；第 10 章汇总全栈联调与排错经验。每章配套独立的可运行源码，读者跟随章节即可在本地从零搭出一套完整的 RAG 应用。

本书面向有一定 Python 基础、希望系统掌握本地大模型工程实践的读者，适合对生成式 AI、Agent 应用、RAG 检索增强、本地部署有兴趣的工程师，也适合作为团队内部技术培训与个人项目实战参考。
