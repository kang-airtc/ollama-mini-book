## 常用命令
常用示例：
### 运行模型（首次会自动下载）
ollama run llama3.2
#### 查看已下载的模型
ollama list
#### 停止运行中的模型
ollama stop llama3.2
#### 删除模型
ollama rm llama3.2
API 调用（默认端口 11434）：
curl http://localhost:11434/api/generate -d '{
  "model": "llama3.2",
  "prompt": "你好"
}'

## RAG流程
client --> 提了一个问题 --> server ---> 查询VectorDB ---> 结果 + Prompt ---> LLM
LLL分析总结-->返回到后端 --> 返回前端

## 项目启动

### 1. 启动 Ollama 服务
```bash
ollama serve
```

### 2. 确保已下载所需模型
```bash
ollama pull llama3.2:latest
ollama pull nomic-embed-text:latest
```

### 3. 启动 Agent (MCP Server)
```bash
cd agent
source venv/bin/activate
python agent_server.py
```

### 4. 启动 Backend (FastAPI)
```bash
cd backend
source venv/bin/activate
uvicorn main:app --reload --port 8000
```

### 5. 启动 Frontend (Next.js)
```bash
cd frontend
npm run dev
```

### 6. 访问应用
浏览器访问: http://localhost:3000
